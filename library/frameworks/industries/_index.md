# Industry snapshots — index

Cheap-to-scan one-liners for the 16 industries the MCCP guide covers. **When a case is set in one of these,
load only that industry's file** (`industries/{slug}.md`) — don't load them all. Each file has the same shape:
overview · key trends · competitive landscape · customers · channels · profit model. Use it as a *prior* to ask
sharper questions and sanity-check the candidate — never as case facts (solve with the case's own data).

| Industry | File | Profit model in one line |
| --- | --- | --- |
| Airlines | [airlines.md](airlines.md) | Ticket + fees revenue; VC fuel/crew, FC aircraft & gate leases; price-driven competition |
| Consumer Packaged Goods | [cpg.md](cpg.md) | Volume × brand price premium; cost = branding + S&M + commodity COGS |
| Banking (consumer) | [banking.md](banking.md) | Net interest spread + fees; cost = overhead, salaries, bad-debt |
| Insurance | [insurance.md](insurance.md) | Spread between premiums and claims; invest the float; cost = admin + commissions |
| Manufacturing | [manufacturing.md](manufacturing.md) | Volume × price; highly cyclical; cost driven by outsourcing/labour |
| Media | [media.md](media.md) | Advertising (+ subscriptions); audience IS the product; VC production, FC capital |
| Metals & Mining | [metals-mining.md](metals-mining.md) | oz mined × market price − cost; can't control price, only AISC; capital-intensive |
| Oil & Gas | [oil-gas.md](oil-gas.md) | Upstream = price × barrels; huge CAPEX; up/mid/downstream split |
| Pharmaceuticals | [pharma.md](pharma.md) | Patent drugs vs. generics; cost = R&D + S&M to doctors; FDA-gated |
| Private Equity | [private-equity.md](private-equity.md) | Returns via financial + operational levers; ROI/timeframe driven |
| Professional Services | [professional-services.md](professional-services.md) | Billable engagements (lump sum or rate × hours); cost = overhead |
| Public Sector (Canada) | [public-sector.md](public-sector.md) | Funded by tax/fees/transfers; "profit" = efficiency + outcomes, not margin |
| Retail | [retail.md](retail.md) | Units × price; cost = COGS, distribution, SG&A, rent; omni-channel shift |
| Technology | [technology.md](technology.md) | Subsector-dependent; software high-margin/R&D, semis cyclical/CAPEX |
| Telecommunications | [telecom.md](telecom.md) | Subscriptions + data; FC = network/infrastructure CAPEX; consolidating |
| Tourism | [tourism.md](tourism.md) | Transport + accommodation services; sharing-economy disruption; ~10% of global GDP |
