# Case Prep

Empty until you run a case. Each practice session gets a folder: `{date}-{case-slug}/`

```
{date}-{case-slug}/
├── session.md      # which case, date, outcome
├── transcript.md   # the full case run
└── debrief.md      # the post-case review (scores + coaching)
```

Start one by saying "run me a case." Answer by **voice** — it's practice for the real interview.
