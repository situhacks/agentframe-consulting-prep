# Applications

Empty until you apply to something. Each job gets a folder: `{company}/`

```
{company}/
├── application.md          # state: company, role, JD (verbatim), stage, status
├── stage-1-research/       # company-research output (optional, "know the firm")
│   └── research-brief.md
└── stage-2-tailor/         # the deliverables
    ├── cv.md
    ├── cover-letter.md
    └── *.docx              # exports
```

Start one by saying "tailor my resume to this JD" (paste the JD) — the `cv` skill creates the folder.

See `EXAMPLE-consulting-analyst/` for a complete worked example (JD → research → tailored CV + cover letter)
using a fictional firm and placeholder candidate. It shows the full pipeline and the quality bar.
